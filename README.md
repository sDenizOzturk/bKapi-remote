## Getting started

- Clone the repository

```
git clone https://github.com/sDenizOzturk/bKapi-remote.git
```

- Install dependencies

```
cd bKapi-remote
npm install
```

- Build and run the project

```
npm run dev
```

Navigate to `http://localhost:5173`
