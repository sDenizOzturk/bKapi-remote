## Getting started

- Clone the repository

```
git clone https://github.com/sDenizOzturk/bKapi-remote.git
```

- Install dependencies

```
cd bKapi-remote
npm install
```

- Build and run the project

```
npm start
```

Navigate to `http://localhost:3000`
